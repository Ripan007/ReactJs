## _How to work with react:_

- Install Node and NPM :

      Node -v
      Npm -v
      Node --version

- Install react from terminal:

  `npx create-react-app <name>`

  `npm install -g create-react-app`

  `create-react-app <projectName>`

  `cd <projectName>`

  `npm start`
